2. Crea progetto → API e servizi → Abilita API → cerca Google Drive API → Abilita

4. Tipo: Applicazione Web → aggiungi http://localhost:8080 nelle Origini JavaScript autorizzate
5. Copia il Client ID (formato: xxxxxxxxxx.apps.googleusercontent.com)
⚠️ Stai usando file:// — Drive richiede un server HTTP.
